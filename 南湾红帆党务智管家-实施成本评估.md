# 南湾红帆党务智管家 — 实施成本评估（不含开发）

> 项目定位：内部管理学习生活工具，面向党务成员，非面向公众
> 用户规模：4000人以内，日活预估数百人
> 运维模式：远程运维为主 + 偶尔驻场，新功能开发另行付费
> 等保：内部工具暂不做等保测评
> 最后更新：2026-04-25

### 前提条件

- 街道办无现成服务器、无固定公网IP
- 直播使用第三方免费平台
- 录播视频约每周1-2份，每份约2小时，年存储增量约400GB
- 现场培训，上线驻场2-3天
- 工作日线上即时响应运维
- 新功能开发不含在运维内，另行付费

## 三、两套方案对比

| 维度 | 方案A：阿里云 | 方案B：本地部署 |
|------|:---:|:---:|
| **首年** | **6.5万** | **8.8万** |
| **3年** | **14.9万** | **17.6万** |
| **5年** | **23.3万** | **26.4万** |
| 硬件投入 | 无 | 1.7万（一次性） |
| 年度采购/服务费 | 1.2万/年 | 0.6万/年 |
| 年度运维费 | 3万/年 | 3.8万/年 |
| 年度持续支出 | 4.2万/年 | 4.4万/年 |
| 数据存放 | 阿里云 | 完全本地 |
| 弹性扩容 | 随时升配 | 需加硬件 |
| 硬件故障 | 云平台兜底 | 自行承担 |
| 运维复杂度 | 低 | 中 |

### 结论

- 阿里云方案全周期更便宜：首年省2.3万，5年省3.1万
- 本地方案唯一优势是数据完全自主可控
- 推荐阿里云方案，除非有数据必须本地存储的硬性要求

## 四、补充说明

1. 以上所有费用**不含开发成本**，开发费用另行评估
2. 新功能开发需求**另行报价**，不含在运维费内
3. 等保测评暂不纳入，如后续上级要求，追加约3万
4. OSS存储费用会随视频量逐年增长，每年约增加400GB，第5年累计约2TB，年存储费用约增至3,000-4,000元
5. DeepSeek API费用按当前V4 Flash定价估算，后续价格可能调整
6. 硬件预计使用寿命5年，5年后可能需要更换，追加约1.5万
