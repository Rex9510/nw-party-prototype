# 复制为 .env 后在启动 shell 中 source / 或自行 setx（Windows）注入环境变量。
# 说明：prototype/mobile_v2.html 中 i龙岗 默认用明文 weixin://dl/business/?appid=… 唤起，一般无需配置下方密钥。
# 仅在页面开启「nw_ilg_use_scheme_api_v1」走同源 /api/wechat/url-scheme（generatescheme）时才需要 AppSecret。
# 微信小程序 URL Scheme（wxa/generatescheme）须使用「被拉起小程序」的 AppID + AppSecret。
#
# i龙岗：AppId 为 wx201f04ef8f8d6b7c（公开）。后端对 ?target=ilonggang 若未设 WECHAT_ILG_APPID / WECHAT_MINI_APPID，将使用该 AppId；
# 仍须在以下任选一对中配置 AppSecret（在微信小程序后台「开发管理 → 开发设置」查看，勿提交到 Git）。
#
# 方式 A（推荐，与 i龙岗 专用 target 对齐）：
WECHAT_ILG_APPID=wx201f04ef8f8d6b7c
WECHAT_ILG_SECRET=在此填写i龙岗小程序AppSecret
#
# 方式 B（与其它 target 共用一对凭证时）：
WECHAT_MINI_APPID=wx201f04ef8f8d6b7c
WECHAT_MINI_SECRET=在此填写同一小程序AppSecret

# 可选：打开后的页面路径与参数（不填则微信按首页规则处理，以微信文档为准）
WECHAT_SCHEME_PATH=
WECHAT_SCHEME_QUERY=

# 当请求参数 target=ilonggang 时优先使用下面两项（便于与通用路径区分；须与小程序线上页面一致）
WECHAT_ILG_SCHEME_PATH=pages/i-longgang/index
WECHAT_ILG_SCHEME_QUERY=

# 可选：scheme 有效天数 1～30，默认 30
WECHAT_SCHEME_EXPIRE_DAYS=30

# 可选：服务端口，默认 8080
# PORT=8080
